# Beemo
Beemo is an antispam Discord bot that allows you to equip your server with a powerful antibot algorithm that operates in the background to stop userbot raids against your server. This is the public repository website built with the help of Sveltekit and Tailwind CSS to bring a beautiful, glazing hive.

You can host this page using [Cloudflare Pages](https://pages.dev) by following the steps provided by Cloudflare @ [Deploying with Cloudflare Pages](https://developers.cloudflare.com/pages/framework-guides/deploy-a-svelte-site#deploying-with-cloudflare-pages).
